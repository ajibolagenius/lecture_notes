Introduction to CSS

CSS stands for Cascading Style Sheets.

CSS describes how HTML elements are to be displayed.

This tutorial will teach you CSS from basic to advanced.

HTML Syntax

HTML uses "elements" to structure the content.

Elements are represented by tags, written using angle brackets.

Most elements have an opening tag and a closing tag.

Some elements are self-closing and do not require a closing tag.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.
