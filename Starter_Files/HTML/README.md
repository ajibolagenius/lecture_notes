# HTML Starter Files

## How to Use

1. Copy the `index.html` file into your project folder
2. Open the folder in VS Code
3. Right-click `index.html` → **Open with Live Server**
4. Your page opens at `http://127.0.0.1:5500`

## What's Included

`index.html` — A valid HTML5 skeleton with:
- `<!DOCTYPE html>` declaration
- `<html lang="en">` root element
- All required `<head>` meta tags:
  - `charset="UTF-8"` — supports all characters
  - `viewport` — correct mobile scaling
  - `<title>` — browser tab text
  - `<meta name="description">` — search engine snippet
- `<link>` to `./css/main.css` — ready for CSS course

## What to Do Next

Replace every placeholder:
- `My Page` → your actual page title
- `A short description...` → a real description (under 160 characters)
- Add your content inside `<body>`

Validate your work at `validator.w3.org` — aim for zero errors.
